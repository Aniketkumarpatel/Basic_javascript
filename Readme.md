# Bootstrap Project

## Overview

This is a small front-end demo project containing a plain HTML page and a JavaScript file. The project loads `index.js` from `index.html` and demonstrates basic JavaScript syntax, arrays, and console output.

## Files

- `index.html` - Simple HTML page that includes the `index.js` script.
- `index.js` - JavaScript file with sample code blocks, commented examples, and a small array example.
- `.git/` - Git repository metadata folder.

## Usage

1. Open `index.html` in a web browser.
2. Open the browser console to view output from `index.js`.

## Notes

- The current `index.js` contains a sample array example and a `console.log(typeof(obe));` statement. Because `obe` is not defined, this will produce a console error unless the variable is defined or the line is removed.
- Most of the content in `index.js` is commented-out example code for learning JavaScript.
